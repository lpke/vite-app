## Commands

- `pnpm test`
- `pnpm typecheck`
- `pnpm format`

## Rules

- **Git: read-only.** NO staging, unstaging, committing, stashing, pushing, pulling, or history changes unless explicitly asked for. Data-reading commands and `git add -N` (intent-to-add) are fine.
- When editing or creating files, run `pnpm format` when you are done with all edits
- Do not kill existing dev servers unless explicitly asked to
